## 配置数据库

1. 打开 `common/configs/typeorm.config.ts`
2. 根据本地环境配置 `databaseConfig` 的数据库信息

## 启动项目

```shell
# 开发调试
> pnpm install # 安装依赖
> pnpm start # 启动开发服务器
```
